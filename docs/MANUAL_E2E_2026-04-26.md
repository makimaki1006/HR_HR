# 手動 E2E チェックリスト — 2026-04-26 リリース

> **対象**: V2 HW Dashboard 2026-04-26 リリース後の **ローカル動作確認**
> **想定所要**: 約 15 分
> **環境**: ローカル `http://localhost:9216` または本番 `https://hr-hw.onrender.com`

本チェックリストは、`tests/e2e/regression_2026_04_26.spec.ts` の自動 E2E が通過した後、人間が UI 上で目視確認するための補完手順です。自動テストでは検出しきれない以下を中心に確認します。

- フォントレンダリング・絵文字・配色 (画像比較を伴う差分)
- HTMX 切替時のチラつき・レイアウトシフト
- 警告バナーの視認性・占有面積
- KPI tooltip の挙動 (hover 時のみ表示されるか)
