# ハローワーク求人分析ダッシュボード 取扱説明書

**最終更新**: 2026年3月19日
**URL**: https://hr-hw.onrender.com

---

## はじめに

### このダッシュボードでわかること

ハローワークインターネットサービスに掲載された求人データを多角的に分析し、**地域の求人市場の「今」と「動き」**を可視化するツールです。

### ⚠️ データの範囲に関する重要な注意

| 注意事項 | 説明 |
|---------|------|
| **対象データ** | ハローワークインターネットサービスに掲載された求人**のみ** |
| **含まれないもの** | Indeed、リクナビ、マイナビ等の民間求人サイトの求人 |
| **データの偏り** | 中小企業・医療福祉・製造業が多い。IT・スタートアップ・外資系は少ない |
| **地域差** | 地方はHW比率が高く実態に近い。都市部は民間サイト併用が多くHWだけでは全貌が見えない |

**このダッシュボードの数値は「HW掲載求人の中での分析結果」であり、求人市場全体を表すものではありません。**

### 外部統計データについて

一部のグラフには「※外部統計データ」と記載されたセクションがあります。これらは以下の公的統計から取得したデータで、HW求人データとは別のデータソースです。

| マーク | 意味 |
|--------|------|
| 出典末尾に「※外部統計データ」 | 国の統計（e-Stat API）から取得。全産業・全チャネルを含む |
| 出典末尾に記載なし | HW掲載求人データから算出 |

**外部統計の有効求人倍率**（厚労省公表値）と**HW求人データから算出した指標**は計算方法が異なります。直接比較する際はこの点にご注意ください。

## タブ別ガイド

### 📊 Tab 1: 地域概況

**このタブで答えられる問い**: 「この地域の求人市場はどんな状態か？」

| セクション | 見るべきポイント | 判断基準 |
|-----------|----------------|---------|
| 求人件数 | 正社員/パートの比率 | パートが多い地域は非正規依存の可能性 |
| 平均給与 | 全国平均との差 | 全国平均より低い→採用難の一因かも |
| 産業分布 | 上位産業の偏り | 1産業が30%超→地域経済がその産業に依存 |
| 雇用形態分布 | 正社員比率 | 50%未満→非正規化が進んでいる地域 |

**注意**: ここに表示される数値はすべてHW掲載求人ベースです。

---

### 🏢 Tab 2: 企業分析

**このタブで答えられる問い**: 「どんな企業が求人を出しているか？」

| セクション | 見るべきポイント | 判断基準 |
|-----------|----------------|---------|
| 企業規模分布 | 従業員数別の求人件数 | 小規模(10人未満)の求人が多い→個人事業主の多い地域 |
| 産業別求人密度 | 産業ごとの求人の集中度 | HHIが高い→少数企業が市場を占有 |
| 法人あたり求人数 | 1法人が出す平均求人数 | 多い→採用に苦労している企業が多い |

---

### 💰 Tab 3: 求人条件

**このタブで答えられる問い**: 「この地域の求人条件の相場はどのくらいか？」

| セクション | 見るべきポイント | 判断基準 |
|-----------|----------------|---------|
| 給与分布 | 中央値と四分位範囲 | 中央値が最低賃金×160h(月額)に近い→最低ライン |
| 年間休日 | 平均と分布 | 105日未満→週休1日相当で労働環境が厳しい |
| 賞与 | 支給率と月数 | 賞与なしが50%超→業界の慣習としてボーナスが少ない |
| 雇用形態別比較 | 正社員vsパートの条件差 | 差が小さい→正社員のメリットが薄い |

**給与について**: HW求人の給与は「基本給+手当」の月額表示が多いですが、時給表示のパート求人も含まれます。比較時は雇用形態フィルタを活用してください。

---

### 📋 Tab 4: 採用動向

**このタブで答えられる問い**: 「企業はなぜ求人を出しているのか？」

| セクション | 見るべきポイント | 判断基準 |
|-----------|----------------|---------|
| 欠員補充率 | 欠員理由の割合 | 30%超→離職が多い可能性。定着支援の営業チャンス |
| 増員率 | 増員理由の割合 | 高い→成長産業。ポジティブな採用 |
| 新設率 | 新規事業所設立の割合 | 高い→地域に新規参入が活発 |
| 募集理由の産業差 | 産業ごとの理由分布 | 介護・飲食で欠員が多い→構造的な人手不足 |

**「未選択」について**: 募集理由が「未選択」の求人が約23%あります。これらは分析から除外されず「理由不明」として扱われます。

---

### 🗺️ Tab 5: 求人地図

**このタブで答えられる問い**: 「求人はどこに集中しているか？」

| 操作 | 説明 |
|------|------|
| ピンの色 | 給与水準や雇用形態で色分け |
| クリック | 求人の詳細情報を表示 |
| ズーム | 市区町村レベルまで拡大可能 |

**注意**: 地図上のピンはジオコーディング（住所→緯度経度変換）の結果です。住所が不正確な求人はピンが表示されない場合があります。

### 📈 Tab 6: 市場分析

**このタブで答えられる問い**: 「市場の構造と外部環境はどうなっているか？」

6つのサブタブがあります。

#### サブタブ 1: 求人動向
欠員補充率、地域レジリエンス（産業多様性）、透明性スコアを表示。

| 指標 | 意味 | 注意 |
|------|------|------|
| 欠員補充率 | 求人理由が「欠員補充」の割合 | HW求人のみの値。全市場の離職率ではない |
| 業界分散度 | HHI（ハーフィンダール指数）の正規化値 | 0.8以上=多様、0.5未満=集中リスク |
| 透明性スコア | 任意項目（年休・賞与等）の記載率 | 低い求人は情報を隠している可能性 |

#### サブタブ 2: 給与分析
給与分布、報酬パッケージ、影の報酬計算器。

| 指標 | 意味 |
|------|------|
| 給与競争力 | 同地域・同産業内での相対順位 |
| 影の報酬 | 基本給+賞与+残業代の推定年収 |
| 最低賃金違反チェック | 時給換算で最低賃金を下回る求人の検出 |

#### サブタブ 3: テキスト分析
求人票の文面から特徴語やテキスト品質を分析。

#### サブタブ 4: 市場構造
充足困難度予測、産業間競合分析。

| 指標 | 意味 | 注意 |
|------|------|------|
| 充足困難度スコア | 0-100で予測（高い=埋まりにくい） | HW掲載求人の特徴量から推定。実際の充足結果ではない |
| グレード分布 A/B/C/D | A=容易〜D=困難 | 予測モデルの結果であり確定的な判断ではない |

#### サブタブ 5: 異常値・外部データ

**最も情報が豊富なサブタブ**です。HW求人データの異常値検出に加え、国の公的統計データを表示します。

| セクション | データソース | 読み方 |
|-----------|------------|--------|
| 異常値検出 | HW求人 | 給与・休日・従業員数で地域平均から大きく外れた求人の割合 |
| 最低賃金マスタ | 厚労省 | 地域の最低賃金。求人給与との比較に |
| 都道府県別外部指標 | 複数の公的統計 | 失業率・転職希望率・非正規率等。**全産業のデータ**でありHW固有ではない |
| 📈 有効求人倍率推移 | 総務省 ※外部 | **全産業の有効求人倍率**の年度推移。HW算出値ではなく国の公式統計 |
| 📊 賃金推移 | 総務省 ※外部 | **全産業の現金給与月額**の推移。男女別 |
| 👥 人口構成 | 国勢調査 ※外部 | 地域の人口・高齢化率・人口ピラミッド |
| 🔄 人口動態 | SSDSE ※外部 | 転入転出・昼夜間人口比 |
| 🏢 産業別事業所数 | 経済センサス ※外部 | 地域の全事業所数（HW求人を出していない企業も含む） |
| 📉 入職率・離職率 | 雇用動向調査 ※外部 | 医療・福祉産業の入職率と離職率。**全チャネルのデータ** |
| 💰 消費支出 | 家計調査 ※外部 | 世帯の消費パターン。福利厚生設計の参考 |
| 🏗️ 事業所動態 | 経済センサス ※外部 | 開業率・廃業率の推移 |
| 🌡️ 気象特性 | 気象統計 ※外部 | 年平均気温・降雪日数等。通勤環境の参考 |
| 🏥 介護需要推移 | 社会統計 ※外部 | 介護保険給付件数の推移。介護求人の先行指標 |
| 🎯 地域ベンチマーク | HW+外部統合 | 12指標で地域を総合評価（0-100） |

**外部データの読み方のコツ**:
- 「※外部統計データ」と書かれたセクションはHW求人とは別の公的統計です
- 外部統計は「全産業・全求人チャネル」を含むため、HW求人データの数値と直接比較できません
- 例：外部統計の有効求人倍率が1.25でも、HW求人から算出した充足率とは一致しません

#### サブタブ 6: 予測・推定
充足困難度の予測モデル結果。

### 🩺 Tab 8: 市場診断

**このタブで答えられる問い**: 「自社の求人条件は市場でどのくらいの競争力があるか？」

| 操作 | 説明 |
|------|------|
| 月給・休日・賞与・雇用形態を入力 | 自社の条件を入力 |
| 「診断する」ボタン | 市場内でのポジションを診断 |
| **「リセット」ボタン** | フォームと結果をクリア（再診断時に使用） |

**診断結果の読み方**:
- レーダーチャートで5軸の相対位置を表示
- 「市場平均より上/下」が一目でわかる
- 具体的な改善提案も表示される

**注意**: 診断結果はHW掲載求人との比較です。民間サイトの求人条件との比較ではありません。

## 指標辞典

### HW求人データから算出される指標

| 指標名 | 定義 | 計算式 | 単位 |
|--------|------|--------|------|
| 欠員補充率 | 求人理由が「欠員補充」の割合 | 欠員求人数 ÷ 全求人数 × 100 | % |
| 増員率 | 求人理由が「増員」の割合 | 増員求人数 ÷ 全求人数 × 100 | % |
| 透明性スコア | 任意開示項目の記載率 | 記載項目数 ÷ 全任意項目数 × 100 | % |
| 充足困難度 | 求人条件から推定した充足の難しさ | 機械学習モデルによる予測（0-100） | 点 |
| 業界分散度 | HHIの正規化値 | 1 - HHI/HHI_max（0-1） | - |
| 掲載期間 | 受付日〜有効期限の日数 | 有効期限 - 受付日 | 日 |

### 外部統計データの指標

| 指標名 | 出典 | 定義 | 注意 |
|--------|------|------|------|
| 有効求人倍率 | 社会・人口統計体系（総務省） | 有効求人数 ÷ 有効求職者数 | **全チャネル・全産業**の値 |
| 完全失業率 | 労働力調査（総務省） | 失業者 ÷ 労働力人口 × 100 | 国勢調査ベース（5年周期の更新） |
| 入職率/離職率 | 雇用動向調査（厚労省） | 入職者/離職者 ÷ 常用労働者 × 100 | 医療・福祉産業のみ表示 |
| 開業率/廃業率 | 経済センサス（総務省） | 新設/廃業事業所 ÷ (存続+廃業) × 100 | 調査間隔（3-5年）の**累計値** |

## ユースケース集

### ケース1: 「営業先を見つけたい」（人材紹介会社向け）

1. **📊 地域概況** で対象地域を選択し、求人件数の多い産業を確認
2. **📋 採用動向** で欠員補充率が高い産業を特定 → 「人が辞めて困っている」企業が多い
3. **📈 市場分析 → 異常値・外部** で入職率・離職率を確認 → HWデータの欠員率と外部統計の離職率が両方高い産業は構造的な人手不足
4. **💰 求人条件** で給与相場を把握 → 「相場より低い給与で出している企業」にコンサルティング提案

### ケース2: 「地域の雇用政策を立案したい」（自治体向け）

1. **📊 地域概況** で正社員比率・平均給与を全国と比較
2. **📈 市場分析 → 異常値・外部** で：
   - 有効求人倍率推移 → 「この地域は全国より高い/低い」
   - 事業所動態 → 「開業率と廃業率のどちらが高いか」
   - 人口ピラミッド → 「生産年齢人口の減少は全国より速いか」
   - 介護需要推移 → 「介護施設の需要は増加傾向か」
3. **🩺 市場診断** で地域の平均的な求人条件を入力し、市場での位置を確認

### ケース3: 「自社の求人条件が適正か知りたい」（事業主向け）

1. **🩺 市場診断** に自社の月給・休日・賞与を入力
2. レーダーチャートで市場平均との比較を確認
3. 改善提案に従って条件を調整
4. **💰 求人条件** で同業他社の条件分布と比較

## よくある質問

### Q: 有効求人倍率が厚労省の公表値と違うのですが？

A: ダッシュボードの「📈 有効求人倍率推移」は**厚労省の公式統計**（社会・人口統計体系経由）をそのまま表示しています。一方、他のタブの指標（充足率等）はHW掲載求人から独自に算出しています。計算方法と対象範囲が異なるため、数値は一致しません。

### Q: 外部統計データはいつ更新されますか？

A: 外部統計データは主に年次更新です。SSDSE-Aは年1回更新、国勢調査は5年に1回です。ダッシュボード上のデータ更新時期は管理者にお問い合わせください。

### Q: パートと正社員を分けて分析できますか？

A: はい。多くのタブでは雇用形態別（正社員/パート/その他）に分けて表示しています。外部統計データのセクションは全雇用形態の合計値で表示されます。

### Q: 特定の企業の求人を探すにはどうすればよいですか？

A: 🔍 詳細検索タブで企業名をキーワード検索してください。

### Q: データは最新ですか？

A: HW求人データはスクレイピング実行時点のスナップショットです。最終更新日はログイン後の画面に表示されます。外部統計データの各出典の年次はセクション末尾の出典に記載されています。

## 用語集

| 用語 | 意味 |
|------|------|
| HW | ハローワーク（公共職業安定所） |
| 有効求人 | 掲載中で有効期限内の求人 |
| 欠員補充 | 退職者の後任として募集する求人 |
| 増員 | 事業拡大のために新たに募集する求人 |
| HHI | ハーフィンダール・ハーシュマン指数（市場集中度の指標） |
| SSDSE | 教育用標準データセット（統計センター提供） |
| e-Stat | 政府統計の総合窓口（API経由でデータ取得） |
| Turso | クラウドデータベース（外部統計データの格納先） |
| スナップショット | ある時点での全有効求人の一括取得データ |
| 複合キー | 事業所番号+職業分類+雇用形態で「同じ募集」を追跡する識別子 |
