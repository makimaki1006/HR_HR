## 1. 目的

`municipality_occupation_population` (市区町村 × 職業大分類 11 区分の推定就業者数) を求めるため、F3 (産業構成補正) を計算する。
F3 計算には「**産業 i の従業者の何 % がどの職業大分類に属するか**」のマッピング (`weight[i, occ]`) が必要となる。

本ドキュメントは、**e-Stat 国勢調査 R2 の産業 × 職業クロス集計 (statsCode 0003450543 系) を取得する前の暫定仮説**として、公知の構成比知識から作成した重みマスタの設計思想・想定効果・限界を記録する。

最終版では本仮説マスタを e-Stat 由来の実測値で置換する。

| 原則 | 内容 |
|------|------|
| **正規化** | 各産業内で `Σ_occ weight[i, occ] = 1.000` (許容誤差 0.001) |

## 4. F3 計算式

F3 (産業構成補正) は、市区町村 `muni` の産業構成に基づき推定された職業 `occ` のシェアを、全国平均で正規化した倍率である。

```
F3[muni, occ] = sigma_i ( industry_share[muni, i] * weight[i, occ] )
                / national_avg_industry_share[occ]
```

ここで、
- `industry_share[muni, i]`: 市区町村 muni における産業 i の従業者シェア (経済センサス R3)
- `weight[i, occ]`: 本マスタ (`occupation_industry_weight.csv`)
- `national_avg_industry_share[occ] = sigma_i ( national_industry_share[i] * weight[i, occ] )`: 全国の産業構成 × 本重みで計算した職業 occ の全国比 (= 正規化分母)

### F3 の解釈

| F3 値 | 解釈 |
|-------|------|
| F3 = 1.0 | 全国平均と同等の職業比 |
| F3 > 1.0 | 当該職業の比が全国平均より高い (例: 工業都市の生産工程職業 F3 = 1.8) |
| F3 < 1.0 | 当該職業の比が全国平均より低い (例: 工業都市の専門技術 (情報通信系) F3 = 0.6) |

`municipality_occupation_population[muni, occ] = sigma_occ_baseline[muni] * F3[muni, occ] * F1 * F2 ...` のように複数補正と組み合わせる。

### 計算例 (建設業集中都市の仮想シナリオ)

ある市の産業構成: D 建設業 18% (全国平均 7%)、E 製造業 15%、その他 67%
- 当該市の建設採掘職業比 = 0.18 × 0.640 + 0.15 × 0.005 + ... ≈ 0.123
- 全国の建設採掘職業比 = 0.07 × 0.640 + 0.16 × 0.005 + ... ≈ 0.052
- F3[muni, 10_建設採掘] = 0.123 / 0.052 ≈ 2.37 (全国平均の 2.37 倍)

## 5. 限界 (Limitations)

| 限界 | 内容 | 影響 |
|------|------|------|
| **県内地域差の不在** | 東京の建設業 (高層・特殊工事の専門技術比が高い) と地方の建設業 (在来工法・10 建設採掘比が高い) を同じ重みで扱う | F3 の県内ばらつきが過小評価される。特に G 情報通信業・L 学術研究で都市部偏差大 |
| **年次固定** | R2 国勢調査の構造を基にしているため、構造変化 (DX・GX・物流再編) を反映しない | 5 年単位で見直し必要 |
| **大分類粒度** | 産業 21 区分・職業 11 区分。中分類 (例: 建設業内の総合工事業/職別工事業/設備工事業) を区別しない | 市内産業集積の細粒度差は捕捉不可 |
| **仮説ベース** | 公知知識からの推定。e-Stat 実測との誤差は ±10〜15 ポイント想定 (各セル) | 主要職業 (主体比) は概ね正しいが、副次職業の配分はズレあり |
| **複合産業の扱い** | Q 複合サービス業 (郵便局・農協) は地域差大 (都市部は金融機能、農村部は営農指導) | 全国平均近似で代用 |
| **X / T (分類不能) の代用値** | 全国平均をそのまま代入 | 該当産業の従業者は全産業平均職業構成として扱われる |

## 6. 検証方法

最終版マスタ作成時に下記の方法で再現性を確認する。

### 6.1 都道府県レベルでの再現性検証

- 国勢調査 R2 の **都道府県 × 職業大分類** 実測値を基準値として取得
- 本マスタを使い、産業構成 (経済センサス R3 都道府県別) から推定した職業大分類シェアを算出
- 47 都道府県で実測値と推定値の RMSE を測定
- 目標: 主要 5 職業 (02 専門技術、03 事務、04 販売、05 サービス、08 生産工程) で RMSE < 3.0pp

### 6.2 市区町村レベルでの妥当性検証

- 市区町村別職業の絶対値は国勢調査でしか公表されないため、**ピアグループ法**で検証
  - 同タイプ市 (例: 工業都市・物流ハブ・住宅都市) を 5〜10 抽出
  - F3 の方向性 (特定職業の浮上) が現場の常識と一致するか確認
- 例: 豊田市 (E 製造業 30%+) → F3[08_生産工程] > 2.0 を期待

### 6.3 端点チェック

- F3 の値域: 想定 0.3 〜 3.0
- F3 < 0.2 または F3 > 5.0 が出る場合は重み or 産業シェアデータの異常を疑う
