# HelloWork ダッシュボード ユーザーマニュアル

**対象**: 営業・採用担当・経営企画
**URL**: https://hr-hw.onrender.com
**最終更新**: 2026-04

---

## 1. はじめに

本ダッシュボードは、ハローワーク(HW)求人データ 469,027件 + SalesNow 企業データ 198,000社 + 国勢調査等の外部統計を統合して、**地域×業種**の採用市場を可視化するツールです。

### 主要な用途

| 用途 | 使うタブ |
|------|---------|
| 地域の求人市場を俯瞰 | 市場概況 / 詳細分析 |
| 地図上で求人を探す | 地図 |
| 競合他社の採用動向を調査 | 企業検索 / 地図（企業マーカー） |
| 自社サイトCSVと市場比較 | 媒体分析 (競合調査) |
| 経営層向け示唆レポート生成 | 市場概況 → 示唆生成 |

---

## 2. ログイン

`test@f-a-c.co.jp` など、事前に払い出されたアカウントでログインします。
セッションは一定時間で失効し再ログインが必要です。

---

## 3. 共通操作: フィルタ

画面上部のヘッダーに以下のフィルタがあります。

| フィルタ | 説明 |
|---------|------|
| 都道府県 | 一つの都道府県を選択。空欄 = 全国 |
| 市区町村 | 都道府県選択後に自動で選択肢が展開 |
| 産業分類 | 大分類→中分類のツリー表示。プリセット (医療介護/IT/建設製造/サービス) あり |

フィルタ変更は全タブに即時反映されます。

## 4. タブ別ガイド

### 4.1 市場概況

都道府県×産業のKPI 4枚 (総求人数 / 平均月給 / 正社員率 / 事業所数) と、雇用形態・年代別グラフ。初見時の全体把握に。

### 4.2 地図

- 市区町村を選ぶと半径10km圏内の求人をマーカー表示
- **人材フロー**パネル: 地域の業種別人材動向 (伸びている業種が上位)
- 業種をクリック → その業種の**上位500社**リスト
- 企業名クリック → 個別プロフィール (SalesNow+HW+外部統計を統合)

💡 **複数企業比較**: 企業リストのチェックボックスで複数選択 → 「選択した企業をCSVダウンロード」ボタンで一括エクスポート

### 4.3 詳細分析

6つのサブタブで深掘り分析:

| Sub | 内容 |
|-----|------|
| 1 構造分析 | 業種×雇用形態の欠員率・増員率マトリクス |
| 2 給与分布 | P25/P50/P75/P90 パーセンタイル統計 |
| 3 テキスト分析 | 求人文からのキーワード抽出 (急募/未経験/待遇/WLB/成長/安定) |
| 4 4象限戦略 | 市場ポジション分類 |
| 5 地域ベンチマーク | 6軸レーダー (求人量/給与/休日/事業所密度/欠員/成長) |
| 6 充足スコア | A-Dグレード分類 |

### 4.4 求人検索

条件フィルタによる求人一覧と、Excel/CSVエクスポート。

### 4.5 条件診断

希望条件を入れると、市場で満たす求人の充足率と代替提案を返します。

### 4.6 企業検索

企業名タイプアヘッドで2文字以上入力。結果をクリックで個別プロフィール。

### 4.7 媒体分析 (競合調査)

Indeed / 求人ボックス等からスクレイピングしたCSVをアップロードし、**自社側データ**として HW と比較します。

**CSV要件**:
- UTF-8 / CP932 自動判定
- 列: 求人タイトル / 企業名 / 勤務地 / 給与 / 雇用形態 / タグ / URL / 新着 (いずれもあれば可)
- 最大20MB
- 「新着」列が無い媒体では KPI 4枚目が **掲載企業数** に自動切替

アップロード後、「統合レポート」ボタンで PDF 化可能。

## 5. レポート生成

### 示唆レポート (/report/insight)

- 1ページ目: エグゼクティブサマリー
- 以降: 22パターンの示唆 (欠員率/給与/採用難易度/成長性/など)
- 印刷ボタン or Chrome印刷 → PDFとして保存

### 競合調査レポート (/report/survey)

- 表紙 + サマリー + HW市場比較 + 給与分布 + 雇用形態 + 散布図 + 地域分析 + 市区町村別給与 + 最低賃金比較 + 企業別 + タグ×給与 + 求職者心理 + **地域注目企業 (SalesNow)**
- A4縦、ECharts SVG レンダリング

### 企業レポート (/report/company/{法人番号})

個別企業の総合診断PDF。営業先資料として。

## 7. よくある質問

**Q. 企業プロフィールが遅い**
A. 初回生成は15〜30秒 (SalesNow/外部統計を直列取得)。2回目以降は15分間キャッシュで200ms以下。

**Q. CSV分析結果が「期限切れ」になる**
A. 分析セッションは約30分で失効。再アップロードしてください。

**Q. 地図マーカーが表示されない**
A. 市区町村の指定が必須です。都道府県だけでは0件になります。

**Q. 375pxスマホ表示が崩れる**
A. 一部の分析テーブルは表幅が1200px超のため、横スクロールで対応してください。

## 8. 既知の制約

- 集計対象はハローワーク掲載求人のみ (全求人市場ではない)
- SalesNow は従業員数 > 0 の企業のみ
- 外部統計は主に国勢調査 2020年ベース (一部2023年更新)
