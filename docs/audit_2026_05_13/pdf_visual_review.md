- 中央領域に箱ひげ図の本体が描画されておらず、左半分はほぼ真っ白

- 表 3-2 統計値テーブルは正常

→ 「箱ひげ図」の ECharts boxplot シリーズが空配列または座標スケール不適合の可能性

### page 08 — 採用市場逼迫度ゲージ

- 「採用難易度 [ゲージ]」セクションのチャート領域が完全に空白
- 直下の「58/100 強 (採用やや困難)」テキストカードは表示されているのにゲージ円弧/針/メーターが描画ゼロ
- ECharts gauge series が描画されていないか、container 高さ 0 の疑い

### page 13 — 図 4-1 雇用形態シェア (ドーナツ/円グラフ)

- セクションタイトル「雇用形態 ドーナツチャート (Top 4)」相当の中央スペースが空白
- 右側に 1.61% / 33.5万円 / 不明 のテキスト凡例が浮いているだけで、ドーナツ円弧が見えない
- 「正社員 (98%)」表記は本文中にあるのにビジュアル化されておらず、KPI 整合が崩れている

### page 14 — 図 5-1 散布図 (月給下限 vs 上限)

- 回帰線 slope = **1,267** (= 12.67 万円/万円幅)、intercept = 21,837。**単位が円ベースのまま slope を計算した可能性**
- 仮に万円ベースなら slope = 0.1267 程度になるはずで、相関係数 r が「(R²=0.0001 等の低値)」と整合しない
- 散布点群はおおむね 30〜40万円帯に密集しているが、回帰線が中央を貫通せず急角度になり、見た目が誤解を招く
- 表 5-1 に「R²: 0.0001 / 解釈: ほぼ無相関」と書いてあり、回帰線描画と数値解釈が矛盾している (ドメイン不変条件違反疑い)

### page 20 — 図 11-2 経験者 vs 未経験可 比較

- バーが 1 本のみ (経験者 34.3411 万円)、未経験可 (35.4 万円) の棒が描画されていない

- 軸の最大スケールがデータ範囲に対して大きすぎる (auto-scale 異常?)
