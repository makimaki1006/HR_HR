## 実装変更

### `src/handlers/survey/report_html/market_intelligence.rs`

`render_mi_market_quadrant` 内、テーブル直前に CSS 散布図ブロックを追加 (~80 行)。

| 要素 | 内容 |
|---|---|
| コンテナ | `position: relative; height: 80mm; border: 1px solid #cbd5e1` |
| 中央線 (median split) | X/Y 各 1 本、`position: absolute` |
| 4 象限ラベル | 4 隅に `position: absolute` (左上=開拓余地 / 右上=重点配信 / 左下=後回し / 右下=条件見直し) |
| 軸ラベル | X 軸下 / Y 軸左 (回転) |
| 各点 | `position: absolute; border-radius: 50%; opacity: 0.8`、サイズ 8〜18px (給与中央値スケール)、色 = 象限ごと |
| 各点ラベル | 自治体名 (12 文字以内) + 求人数 |
| 凡例 | 散布図下、4 色 + 円サイズ説明 |

### 軸スケール

両軸とも **対数スケール** (`f64::ln`):
- X 軸: `10% + 80% * (ln(c+1) - ln_min) / (ln_max - ln_min)`
- Y 軸: `90% - 80% * (ln(e) - ln_min) / (ln_max - ln_min)` (上下反転)

特別区部 (8M 人) と伊達市 (1万人) のレンジ差約 800 倍を線形スケールで表すと小規模自治体が潰れるため、対数で吸収。

### 色マップ

| 象限 | 色 (hex) |
|---|---|
| 重点配信 | `#1e3a8a` (濃紺) |
| 条件見直し | `#c2410c` (橙) |
| 開拓余地 | `#15803d` (緑) |
| 後回し | `#64748b` (灰) |

象限ラベル (4 隅) も同色を使用 (淡色なし、視認性優先)。

[散布図 80mm]

  │開拓余地             │     重点配信       │
  │                     │                    │
  │              北海道 │         ●(8M, 17件)│

                ↑ median 縦線
凡例: ● 重点配信 ● 条件見直し ● 開拓余地 ● 後回し / 円サイズ: 給与中央値の相対値 (大 = 高給与)。両軸は対数スケール。
